==元件模型==（等值Y接，单相等值） + 发电机：恒定电源 + 线路、变压器：$\Pi$型等值电路 + 负荷：恒功率、恒电流、恒阻抗或三者结合 ==系统模型==：根据单线图，连接元件等值电路，得电力系统等值电路 #### 电力系统标么（幺）制 + 有名值：有具体物理单位的量值 + 标幺值：无具体物理单位的相对值。有名值（单位）/选定基值（同单位） + 标幺值特点： + 无量纲 + 不唯一：随基值的变化而变化，须指明基值。为简化计算，基值的选取须遵循规律。 + 单相电路 + 四个模量：电压U~P~、电流I~P~、功率S~P~、阻抗Z + 四个基值：U~PB~、I~PB~、S~PB~、Z：基值满足约束条件，标幺值公式与有名值相同 $\left\{\begin{array}{c}U_P = I_PZ\\S_P = I_PU_P\end{array}\right.$如选：$\left\{\begin{array}{c}U_{PB} = I_{PB}Z_B\\S_{PB} = I_{PB}U_{PB}\end{array}\right.$(约束条件) 有：$\left\{\begin{array}{c}U_{P*} = I_{P*}Z_*\\S_{P*} = I_{P*}U_{P*}\end{array}\right.$ + 有2个约束条件：4个基值中自由度为2，一般选取功率、电压的基准值==S~PB~、U~PB~==有$I_{PB} = S_{PB}/U_{PB} $ $Z_{PB} = U^2_{PB}/S_{PB}$ + 复阻抗：$\dot{Z} = Z\angle\varphi = R + jX$ 令R~B~=X~B~=Z~B~，则：$\dot{Z}_*\triangleq \frac{Z\angle\varphi}{Z_B} = Z_*\angle\varphi = R_* + jX_*$(==与有名值保持一致==) + 复功率：$\dot{S}_P = P_P + jQ_P$ 令$P_{PB} = Q_{PB} = S_{PB}$，则：$\dot{S}_{P*}\triangleq \frac{\dot{S}_P{}}{S_{PB}} = P_{P*}+jQ_{P*}$ (==与有名值保持一致==) + 对称==三相==基值选取规律（Y接法） + 四个模量：U~L~、I~L~、S~3~、Z + 给出得电压、电流不做说明就是线值 + 给出的功率S不做说明就是三相功率 + 给出的阻抗不做说明就是==单相阻抗== + 接法不做说明就是Y解法 + 四个基值：U~LB~、I~LB~、S~3B~、Z~B~ $\left\{\begin{array}{c}U_L = \sqrt{3}I_LZ=\sqrt{3}U_P\\S_3 =\sqrt3 U_LI_L=3U_PI_P=3S_P\end{array}\right.$如选：$\left\{\begin{array}{c}U_{LB} = \sqrt{3}I_{LB}Z_B=\sqrt{3}U_{PB}\\S_{3B} =\sqrt3 U_{LB}I_{LB}=3U_{PB}I_{PB}=3S_{PB}\end{array}\right.$（约束条件）有：$\left\{\begin{array}{c}U_{L*} = I_{L*}Z_*=U_{P*}\\S_{3*} = U_{L*}I_{L*}=U_{P*}I_{P*}=S_{P*}\end{array}\right.$(与单相保持一致) + 基值满足约束条件：三相公式和单相一致 + 线电压标幺值 = 相电压标幺值（无$\sqrt3$） + 三相功率标幺值 = 单相功率标幺值（无3） + 有2个约束条件：4个基值中自由度为2，一般选取功率、电压的基准值==S~3B~、U~LB~==有 $I_{LB} = \frac{S_{3B}}{\sqrt3U_{LB}}$，$Z_B = \frac{U_{LB}}{\sqrt3I_{LB}}=\frac{U^2_{LB}}{S_{3B}}$ + 对称三相电路中，每相阻抗仍为$\dot{Z}_*=R_*+jX_*$ + 三相复功率：$\dot{S}_3 = P_3 + jQ_3$ 令$P_{3B} = Q_{3B} = S_{3B}$，则$\dot{S}_{3*}\triangleq \frac{\dot{S}_3{}}{S_{3B}} = P_{3*}+jQ_{3*}$， $\dot{S}_{3*}=\dot{S}_{P*}=\dot{U}_{P*}\dot{I}_{P*}^*$ 用相电压、相电流标幺值算出的功率标幺值，即代表单相，也代表三相。 + 多元件组成的电路：==基值要统一== + 变压器铭牌参数与标幺值的关系(以变压器额定值S~N~、U~N~为基准)： + $G_{T*}=\Delta P_{0*}$ + $B_{T*}=I_{0*}$ + $R_{T*}=\Delta P_{s*}$ + $X_{T*}=U_{s*}$ + 基值变化对标幺值的影响 + 同一电路中 ，基值应统一 + 设备的标幺值：以==各自额定值==为基准，不统一 + 基值变化时，标幺值换算原则：==有名值不变原则== + 多电压等级电力系统的标幺值 3种做法(2中精确，1种近似) + 逐级归算法（精确） + 各选电压法（精确，常用） + 标准变比变压器和非标准变比变压器，变压器的变比标幺值：$K_* = \frac{U_{t1*}}{U_{t2*}}=\frac{\frac{U_{t1}}{U_{B1}}}{\frac{U_{t2}}{U_{B2}}} =
