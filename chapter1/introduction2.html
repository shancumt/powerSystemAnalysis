确定额定电压

  • 标准化

  • 最佳的技术经济性能

  • 线路(电网)额定电压=用电设备额定电压

  • 发电机额定电压=1.05线路额定电压

  • 升压变压器额定电压：

    • 一次侧 = 发电机额定电压（与发电机相联：1.05）=线路额定电压（与线路相联）
    • 二次侧 = 1.10线路额定电压

  • 降压变压器额定电压：

    • 一次侧 = 线路额定电压

    • 二次侧 = 1.05（或1.10）线路额定电压

  • 分解功率

    • 有功功率：

      由与产生的瞬时功率，周期为，始终0，均值，是无源网络中等效电阻的耗能效率。

    • 无功功率：

      是与产生的瞬时功率，周期为，均值=0，峰值sinφ，是无源网络中等效电抗与外电源的电能量交换速率（物理意义）。

    • 有功功率P定义为的均值（特征值）：

    • 无功功率Q定义为峰值（特征值）：

    • P物理意义：无源网络消耗的功率均值。

    • Q物理意义：无源网与外界交换能量能力的大小，交换功率峰值，均值为零，实际未消耗，描述无源网电感和电容存储和释放场能现象。

    • 无功负荷：网络若是电感性的，则Q>0，习惯上认为网络吸收感性无功。

    • 无功电源：网络若是电容性的，则Q<0，习惯上认为网络发出感性无功。

      吸收感性（或容性）无功功率相当于发出容性（或感性）无功功率

电力系统负荷

  • 什么是电力系统负荷

    电力系统负荷是电力用户所需电功率总和，也称电力系统综合用电负荷。可能包括工农业、交通运输和人民生活等各方面的各种用电设备。

  • 电力系统负荷间的关系图

    • 厂用电：维持发电厂运行自身所需的功率
    • 网络损耗（网损）：电能传输、分配中消耗的功率

  • 负荷曲线

    • 描述描述负荷变化趋势的数学手段：指一段时间内负荷随时间变化的曲线，可用来预测负荷趋势。

    • 分类

      • 负荷类型

        • 有功
        • 无功

      • 时间

        • 日、周、旬、月、年

      • 对象

        • 发电厂、变电所、线路、用户
